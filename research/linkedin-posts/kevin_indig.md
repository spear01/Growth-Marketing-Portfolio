## Post 1
Good prompt tracking starts with sample design.

For your average B2B SaaS CRM category, start with ~40 seed prompts:
12 brand, 12 category, and 16 problem prompts. The problem prompts 
matter because that is where purchase intent usually lives.

Then run each prompt 5x per platform every week. ChatGPT, Perplexity, 
Gemini, and Google AI Overviews should each get their own score because 
aggregation hides how differently the engines behave.

The panel also needs personas. CFO, IT, and marketing buyers evaluate 
the same CRM category through different criteria, so averaging them 
together hides where a brand actually wins or loses.

Metric set should include:
- Mention rate
- Citation rate
- Average position
- Sentiment
- Attributes attached to each mention (all with confidence intervals)

Key insight: Prompt tracking becomes more accurate when each answer 
becomes one sample in a designed panel, not one truth in a dashboard.
